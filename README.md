# Task 01: Excel Sales Dashboard

Created an interactive **Sales Dashboard** using **Microsoft Excel** to analyze the Global Superstore dataset and generate business insights.

---

## 📌 Project Overview

The objective of this project was to clean, organize, and visualize sales data using Excel. The dashboard provides an easy-to-understand overview of sales performance, profitability, and regional trends to support business decision-making.

---

## ✨ Features

- Sales by Category
- Profit by Category
- Sales Trend Over Time
- Region-wise Sales Analysis
- Top 10 Products by Sales
- KPI Cards 
- Interactive Dashboard
- Business Insights

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- KPI Cards

---

## 📈 Key Insights

- Technology category recorded the highest sales.
- Technology generated the highest profit.
- Sales showed consistent growth from 2011 to 2014.
- Apple Smart Phone emerged as the top-selling product.
- Central region contributed the highest overall sales.
